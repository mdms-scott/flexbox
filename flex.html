The Flexible Box Module

Formatting a page made easy through use of the flexible box module. Designed expressly with the intent of laying out webpages, flexible boxes are great for dynamic content.

The flexibility

Despite the section elements having the same specified width, this section is quite a bit larger. In the section to the left, box-flex is set to 0, thus the box will keep its specified size. When box-flex is set to 1, as in the case of this section, the element will be resized to occupy the remaining space. Also, because of the nature of the flexible box, despite there being different amounts of content in the two sections, the columns are automatically kept to the same vertical height without any additional specification.

Ordering of sections

Despite the fact that this is the second section in order by the html, it is displayed first here through specification in the stylesheet. The components of the article (the sections) can be arranged in any order one wishes.
